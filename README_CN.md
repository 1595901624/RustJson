一个 Json 转 Rust Struct 的 Idea 插件（开发中）

## 特性

将 Json 转换为 Rust Struct。
插件仍在开发中...

* 支持 `debug` 派生
* 支持 `serde` 派生
* 支持 `serde` 重命名

## 使用方法

1. 克隆并构建项目
2. 从本地磁盘安装插件
3. 打开一个 json 文件
4. 按 `Alt + Q` 打开窗口或右键单击打开上下文菜单并选择 `Generate Rust Struct`
5. 输入 Json 文本并按 `OK`
